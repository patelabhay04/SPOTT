# 🎟️ SPOTT - AI Event Organiser

![Next.js](https://img.shields.io/badge/Next.js-16-black)
![React](https://img.shields.io/badge/React-19-blue)
![Convex](https://img.shields.io/badge/Convex-Backend-purple)

![Vercel](https://img.shields.io/badge/Deployed-Vercel-black)

> Developed by **Abhay Patel**

SPOTT is an AI-powered event management platform that enables users to create, discover, and manage events seamlessly using OpenRouter AI, Clerk Authentication, and Convex.

## ✨ Features

* AI-powered event generation
* Secure user authentication
* Event creation and management
* Location-based event discovery
* Ticket management system
* Responsive modern UI

---

## 🛠️ Tech Stack

* Next.js 16
* React 19
* Tailwind CSS
* Convex
* Clerk
* OpenRouter AI
* Unsplash API
* Vercel

---

## ⚙️ Environment Variables

```env
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
CLERK_JWT_ISSUER_DOMAIN=

NEXT_PUBLIC_UNSPLASH_ACCESS_KEY=

OPENROUTER_API_KEY=
```

## 🚀 Run Locally

```bash
git clone https://github.com/patelabhay04/SPOTT.git
cd SPOTT
npm install
npm run dev
```
